# Glossary
**Thought**  
A blog post.

# Shortcodes
### Large block image:
\{\{< post-image "/path/to/image" ext >}}

#### Two files should be present:
/path/to/image.ext  
/path/to/image@2x.ext

#### Example
\{\{< post-image "/infinite-cranes-1/crane-tessellation-3d-grid" png >}}
